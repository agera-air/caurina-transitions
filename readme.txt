/as2       : Tweener source classes, Actionscript 2 version. Must be used on AS2 projects (from Flash 8 on).
/as3       : Tweener source classes, Actionscript 3 version. Must be used on AS3 projects (from Flash 9 on, and Flex).
/as2_fl7   : Tweener source classes, Actionscript 2 version, *without* advanced features such as filters. Must be used on AS2 projects (from Flash 7 on, and from Flash Lite 2 on).
/docs      : Documentation, in various shapes and forms (or almost, since it's still being written).
/examples  : Example files with source.
